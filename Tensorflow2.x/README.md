# tf2-train.ipynb
通过读取csv中的空调数据建立空调温度预测模型，并将数据模型保存为json格式，方便使用GO语言或者C++在服务器端进行模型读取.
通过tensorflowlite将模型转换为可以供C语言读取的模式.
使用DNN网络.
最后对比python模型和导出成C版本后模型预测的区别.
